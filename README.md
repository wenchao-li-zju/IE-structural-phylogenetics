# Data and code for "Genuine but unstable: the phylogenetic 
signal of corpus-derived structural metrics in Indo-European"

Wenchao Li & Haitao Liu

## Files
- `metrics75.csv` — 16 languages × 75 structural metrics (raw values)
- `AppendixE_IE_74.csv` — Pagel's λ and Blomberg's K for the 74 tested metrics
- `IE_relaxed19_B4.nex`, `IE_strict4_B4.nex` — discretized character matrices (MrBayes/IQ-TREE input)
- `relaxed-B3.zip`, `relaxed-B4.zip`, `relaxed-B5.zip` — MrBayes runs (consensus trees, convergence diagnostics) at three discretization levels
- `ML_relaxed17_results.zip` — IQ-TREE maximum-likelihood results

## Software
R 4.x (ape, phytools, geiger, phangorn); MrBayes 3.2.7; IQ-TREE 2

## Data sources
UD treebanks v2.15 (universaldependencies.org); IE-CoR (Anderson et al. 2025)
